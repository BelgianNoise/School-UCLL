Wat doet deze code?

product(n)

Schrijf een recursieve functie product(n) die het product van 1 tot en met n uitrekent. Ga ervan uit dat n positief is.

sum(a, b)

Schrijf een recursieve functie sum(a, b) die de som van a tot en met b uitrekent.

euclid(a, b)

Schrijf een functie euclid(a, b) die gegeven twee gehele getallen a en b uitrekent wat hun grootste gemene deler is. Gebruik hiervan het algoritme van Euclides: \[ \mathrm{ggd}(a, b) = \left\{ \begin{array}{lr} a & b = 0 \\ \mathrm{ggd}(b, a \;\mathrm{mod}\; b) & b \neq 0 \\ \end{array} \right. \]

range(a, b)

Schrijf een recursieve functie range(a, b) die gegeven twee getallen a en b de array [a, a+1, a+2, …, b] opbouwt.

double(xs)

Schrijf een recursieve functie double(xs) die gegeven een array getallen xs een nieuwe array teruggeeft waarvan de elementen gelijk zijn aan het dubbel van hun overeenkomstige element in xs.

contains(xs, x)

Schrijf een recursieve functie contains(xs, x) die, gegeven een array xs en een waarde x, nagaat of x voorkomt in xs.

indexOf(xs, x)

Schrijf een recursieve functie indexOf(xs, x) die, gegeven een array xs en een waarde x, de kleinste index van x in xs teruggeeft. Indien x niet voorkomt in x, moet de functie -1 teruggeven.

isPalindrome(str)

Schrijf een recursieve functie isPalindrome(str) die, gegeven een string str, nagaat of het een palindroom is. Een string is een palindroom indien men als men de letters omkeert, men dezelfdstring uitkomt. Voorbeelden zijn non, lepel, negen, etc.

digits(n)

Schrijf een recursieve functie digits(n) die, gegeven een positief geheel getal n, de cijfers van n teruggeeft als een lijst getallen.

remove(xs, x)

Schrijf een recursieve functie remove(xs, x) die, gegeven een array xs en een waarde x, een nieuwe array teruggeeft die gelijk is aan xs waaruit alle voorkomens van x werden verwijderd.

removeDuplicates(xs)

Schrijf een recursieve functie removeDuplicates(xs) die, gegeven een array xs, een nieuwe array teruggeeft gelijk aan xs waaruit alle duplicaten werden gehaald. Indien een element meermaals voorkomt, mag enkel het meest linkse element voorkomen in de resulterende array.

reverse(xs)

Schrijf een recursieve functie reverse(xs) die de gegeven array xs omkeert. De functie moet de gegeven array zelf niet wijzigen, maar een nieuwe array teruggeven die de geïnverteerde vorm van xs is.

money(xs)

Je beschikt over een aantal munten en briefjes. Je wil weten of je een gegeven bedrag gepast kunt vormen. Schrijf een functie money(coins, goal) die, gegeven een array getallen coins die de muntstukken/briefjes voorstellen en een getal goal dat het doelbedrag voorstelt, true teruggeeft indien je het bedrag exact kunt vormen, en false indien niet.

reachable(map, from, to)

Je krijgt een landkaart. Elke stad stellen we voor door een index, d.i. steden 0, 1, 2, 3, … Steden kunnen verbonden zijn met elkaar via wegen, maar uiteraard is niet elke stad verbonden met elke andere stad.

We stellen de landkaart voor als volgt: per stad sommen we de rechtstreeks bereikbare steden op. Stel dat steden 4, 7 en 8 bereikbaar zijn, dan schrijven we dit als de array [4,7,8]. Dit herhalen we voor elke stad: we krijgen dus een array van lijsten bereikbare steden.

Bijvoorbeeld: beschouw een landkaart met 3 steden op, genummerd 0, 1 en 2. Vanuit stad 0 vertrekt een éénrichtingsweg naar 1, vanuit 1 is er een éénrichtingsweg naar 2. Dit schrijven we als [[1], [2], []]. Op index 0 staat [1], wat betekent dat men vanuit stad 0 stad 1 rechtstreeks kan bereiken. Op index 2 staat [], wat aangeeft dat men onmogelijk stad 2 kan verlaten.

Schrijf een functie reachable(map, from, to) die gegeven een landkaart map, een vertrekpunt from en een eindpunt to, nagaat of men vanuit het vertrekpunt het eindpunt kan bereiken.

countdown(ns, goal)

In het Britse spelprogramma Countdown (dat in Vlaanderen gekend is onder de naam "Cijfers en Letters") krijgen kandidaten 6 positieve gehele getallen en een doelgetal. De bedoeling is dat ze met deze 6 gegeven getallen via berekeningen (optelling, aftrekking, vermenigvuldiging en deling) het doelgetal kunnen uitkomen.

Bijvoorbeeld, stel dat je de getallen 8, 4, 4, 6, 8 en 9 krijgt, en het doelgetal is 594. De volgende berekening toont aan hoe je dit doelgetal kan bereiken:

  • 8 + 8 = 16
  • 16 * 4 = 64
  • 6 - 4 = 2
  • 64 + 2 = 66
  • 66 * 9 = 594

Tijdens een berekening mogen enkel positieve gehele getallen voorkomen: negatieve of kommagetallen zijn dus geen geldige tussenresultaten.

Schrijf een functie countdown(ns, goal) die gegeven een array positieve gehele getallen en een doelgetal goal nagaat of het mogelijk is om het doelgetal te bekomen.
