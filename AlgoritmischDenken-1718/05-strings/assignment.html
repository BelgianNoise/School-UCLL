Strings

Een string wordt in code voorgesteld d.m.v. aanhalingstekens. Zo kan men de naam Jan schrijven als "Jan". Het gebruik van apostrofes is ook toegelaten: 'Jan' en "Jan" zijn equivalent.

Als str een string bevat, dan heb je toegang tot de volgende functionaliteit:

Syntax Beschrijving
str.length Lengte van de string.
str.charAt(i) Geeft het teken met index i terug.
parseInt(str) Zet string om naar geheel getal.
parseFloat(str) Zet string om naar getal.
str.concat(str2) Samenvoegen tot nieuwe string. Kan ook geschreven worden als str + str2.
str.includes(str2) Gaat na of str2 een deelstring is van str.
str.startsWith(str2) Gaat na of str2 een prefix is van str.
str.endsWith(str2) Gaat na of str2 een suffix is van str.
str.split(sep) Split de strings op met sep als separator.
str.substr(i, len) Deelstring beginnende bij index i en len lang.

Meer informatie kan je online vinden.

Merk op dat er een verschil is tussen 123 en "123": de eerste waarde is een getal, terwijl de tweede een string is. Als je 123 + 456 uitrekent, krijg je 579. Als je echter "123" + "456" evalueert, geeft dit "123456".

De operator + kan dus gebruikt worden op zowel getallen en strings. Dit "recycleren van eenzelfde operator" heet overloading. Overloaden betekent dat er meer dan één betekenis gegeven wordt aan eenzelfde symbool. Dit concept komt in veel programmeertalen voor.

Je kan je afvragen wat er gebeurt als je een string met een getal optelt, bijvoorbeeld "123" + 456. Javascript zal het getal eerst omzetten naar een string: 456 wordt "456". Vervolgens wordt concatenatie toegepast en krijg je "123456" als resultaat.

JavaScript ondersteunt tevens string interpolation: dit laat je toe gemakkelijk strings op te bouwen. Stel dat je variabelen d = 1, m = 5 en y = 2000 hebt, dan zal `${d}/${m}/${y}` de string "1/5/2000" produceren.

Code Points

Intern werkt een computer uitsluitend met bits. Deze kunnen gebruikt worden om alle soorten informatie voor te stellen: getallen, muziek, beelden, tekst, etc.

Gehele getallen worden voorgesteld door een aantal bits te groeperen (typisch 8, 16, 32 of 64) en deze te interpreteren als de binaire voorstelling van een getal.

Tekst is een sequentie van tekens zoals letters, cijfers en interpunctie. Om zo een teken voor te stellen moet de computer weer gebruik maken van bits. Het principe is eenvoudig: er werd een gigantische tabel gedefinieerd waar elk teken een eigen nummer krijgt, code point genoemd. Enkele voorbeelden:

Code PointTekenCode PointTekenCode PointTeken
65A 97a 480
66B 98b 491
67C 99c 502

Hier kan je een volledig overzicht vinden.

Om gegeven een codepoint n het overeenkomstige teken op te vragen, kan je String.fromCharCode(n) gebruiken. Omgekeerd, om de code point van een teken op te vragen, plaats je het teken in een string str en roep je str.charCodeAt(i) op, met i de index van het teken in str.
