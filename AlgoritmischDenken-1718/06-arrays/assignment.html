Je kan hiermee het gemiddelde uitrekenen van vier willekeurige getallen. Wat echter als je vijf getallen hebt? Moeten we dan een nieuwe functie schrijven met vijf parameters? Moeten we telkens hetzelfde doen voor elk mogelijk aantal getallen?

Om dit probleem op te lossen, biedt JavaScript arrays aan. Een array kan je zien als een lijst van variabelen. Zo kan je een array van 10 lang maken en kan je hier 10 waarden (getallen, booleans, strings, of zelfs andere arrays) in bewaren.

Een array kent posities toe aan zijn elementen. Zo kan men aan een array vragen wat zijn eerste, tweede, etc. element is. Dit gebeurt d.m.v. een volgnummer of index. Het eerste element heeft index 0, het tweede index 1, etc. Let dus op dat je bij 0 begint te tellen en niet bij 1.

De basisoperaties op arrays zijn:

  • Aanmaak van een array van lengte n gebeurt met new Array(n).
  • Vaak krijg je een array van buitenaf en zal je willen weten hoeveel elementen er in de array opgeborden zijn. Het aantal elementen vraag je met xs.length, waarbij xs de variabele is die de array bevat.
  • Indexeren: gegeven een array xs kan je het element met index i opvragen d.m.v. xs[i]. Let op dat de index geldig is: het eerste element heeft index 0 en het laatste element xs.length - 1. Enkel indices tussen deze twee extremen zijn geldig. Vraag dus niet het 100ste element van een array van 10 lang. Deze fout wordt vaak gemaakt en spijtig genoeg zal JavaScript hier geen duidelijke foutmelding over geven!
  • Schrijven naar een array gebeurt met een gelijkaardige syntax: xs[i] = waarde. Deze code overschrijft het element met index i met de waarde.

Extra Operaties

Arrays in JavaScript bieden tal van hulpoperaties, waaronder

  • concat laat toe arrays samen te voegen: xs.concat(ys) geeft een nieuwe array terug waarvan de elementen gelijk zijn aan die van xs gevolgd door die van ys.
  • xs.pop() verwijdert het laatste element en geeft dit terug als resultaat. Dit wijzigt m.a.w. de lengte van de array.
  • xs.push(x) voegt x toe achteraan de array.
  • xs.shift() verwijdert het eerste element en geeft dit terug als resultaat.
  • xs.unshift(x) voegt een waarde x toe aan het begin van de array xs.
  • xs.slice(i, j) kopieert de elementen met indices i, i+1, …, j-1 naar een nieuwe array. Merk op dat het element met index j zelf niet mee gekopieerd wordt.

transfer(xs, ys)

Schrijf een functie transfer(xs, ys) twee arrays xs en ys ontvangt.

  • Indien xs niet leeg is, wordt het laatste element uit xs verwijdert en dit plaats vooraan in ys. De functie moet dan true teruggeven.
  • Indien xs leeg is, moet ys ongewijzigd blijven. De functie moet in dit geval false teruggeven.

split(xs)

Schrijf een functie split(xs) die de gegeven array xs splitst in twee deelrijen van "ongeveer gelijke" lengte. We noemen deze twee deelrijen left en right.

  • Indien xs een even aantal elementen telt, moeten left en right even lang zijn.
  • Indien xs een oneven aantal elementen bevat, moet left één langer zijn dan right.
  • left en right samenvoegen moet weer xs opleveren.
  • Vermits functies in Javascript telkens maar één waarde kunnen teruggeven, moeten left en right samengebundeld worden in een nieuwe array die dan fungeert als returnwaarde.

Schrijf een functie missingNumber(ns) die het ontbrekende getal in de array getallen ns teruggeeft. ns is verondersteld alle getallen van $1$ t.e.m. $n$ te bevatten. longestIncreasingSubsequence(ns)

Schrijf een functie longestIncreasingSubsequence(ns) die gegeven een array van getallen xs zoekt naar de langste stijgende deelrij. Indien er meerdere zulke deelrijen bestaan, wordt de eerst voorkomende teruggegeven. Zo bevat [5, 6, 7, 1, 2, 3] twee langste deelrijen, nl. [5, 6, 7] en [1, 2, 3]. In dit geval moet de functie [5, 6, 7] teruggeven.

Met deelrij bedoelen we een ononderbroken reeks elementen van xs: zo is [1, 2, 3] een deelrij van [0, 1, 2, 3, 4], maar niet van [1,0,2,0,3].

Gelijke elementen beschouwen we als stijgend: zo is [1, 1, 2, 2, 3, 3] een stijgende rij.

mastermind(secret, guess)

Mastermind bestaat erin dat je een geheime code moet weten te vinden. Deze code bestaat uit $N$ cijfers. Je mag hiervoor zelf meermaals gokken. Voor elke gok krijg je feedback in de vorm van twee getallen:

  • Het eerste getal geeft aan hoeveel cijfers je correct gegokt hebt.
  • Het tweede getal is het aantal cijfers die wel voorkomen in de code, maar op een andere plaats.

Stel dat de code 1 2 3 4 is en je gokt 1 5 3 2. De feedback die je hierop krijgt is 1 2:

  • Het eerste cijfer 1 is juist en staat op de juiste positie.
  • De cijfers 2 en 3 komen in de code voor, maar in je gok staan ze op de vierde en derde plaats, respectievelijk, terwijl ze op de tweede en derde plaats horen.

Schrijf een functie mastermind(secret, guess) die, gegeven twee arrays getallen secret en guess, de feedback teruggeeft als een array van twee getallen.
