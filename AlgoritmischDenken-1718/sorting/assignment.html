Sorting

Wat doet deze code?

Bubble Sort

swap(xs, i, j)

Schrijf een functie swap(xs, i, j) die, gegeven een array xs en twee gehele getallen i en j, de array xs aanpast zodat de elementen met index i en j van plaats verwisseld zijn.

Je mag ervan uitgaan dat i en j geldige indices zijn.

bubbleSort(ns)

Schrijf een functie bubbleSort(ns) die de gegeven array getallen sorteert gebruik makende van het bubble sort algoritme.

Selection Sort

indexOfMinimum(xs)

Schrijf een functie indexOfMinimum(ns) die, gegeven een array van getallen, de index van het kleinste element teruggeeft. Indien er meerdere zulke elementen zijn, moet de kleinste index teruggegeven worden. Indien ns leeg is, moet het resultaat -1 zijn.

removeAt(xs, index)

Schrijf een functie remoteAt(xs, i) die, gegeven een array xs en een getal i, een nieuwe array teruggeeft gelijk aan xs waarbij het element met index i werd verwijderd. De array xs zelf mag dus niet gewijzigd worden.

Je mag ervan uitgaan dat i een geldige index is voor xs.

selectionSort(ns)

Schrijf een functie selectionSort(ns) die de gegeven array getallen sorteert gebruik makende van het selection sort algoritme.

Insertion Sort

insertSorted(ns, n)

Schrijf een functie insertSorted(ns, n) die, gegeven een gesorteerde array getallen ns en een getal n, een nieuwe array teruggeeft gelijk aan ns waarbij n werd ingevoegd zodat het resultaat gesorteerd is.

insertionSort(ns)

Schrijf een functie insertionSort(ns) die de gegeven array getallen sorteert gebruik makende van het insertion sort algoritme.

QuickSort

split(ns, n)

Schrijf een functie split(ns, n) die, gegeven een array getallen ns en een getal n, de array ns opsplitst in twee partities, nl. enerzijds de elementen kleiner dan of gelijk aan n, en anderzijds de elementen groter dan n. De onderlinge volgorde van de elementen moet behouden blijven.

Het resultaat geef je terug als een array van twee arrays: [left, right], waarbij left alle kleinere en right alle strikt grotere elementen gevat.

quickSort(ns)

Schrijf een functie quickSort(ns) die de gegeven array getallen sorteert gebruik makende van het quicksort algoritme.

Merge Sort

merge(ks, ns)

Schrijf een functie merge(ks, ns) die, gegeven twee gesorteerde arrays getallen, een nieuwe gesorteerde array teruggeeft die alle elementen uit ks en ns bevat.

mergeSort(ns)

Schrijf een functie mergeSort(ns) die de gegeven array getallen sorteert gebruik makende van het merge sort algoritme.

Counting Sort

minimum(ns)

Schrijf een functie minimum(ns) die gegeven array getallen ns het kleinste element teruggeeft. Je mag ervan uitgaan dat ns altijd minstens een element bevat.

maximum(ns)

Schrijf een functie maximum(ns) die gegeven array getallen ns het grootste element teruggeeft. Je mag ervan uitgaan dat ns altijd minstens een element bevat.

zeros(n)

Schrijf een functie zeros(n) die, gegeven een positief geheel getal n, een array teruggeeft van lengte n, gevuld met 0-waarden.

countingSort(ns)

Schrijf een functie countingSort(ns) die de gegeven array getallen sorteert gebruik makende van het counting sort algoritme.
